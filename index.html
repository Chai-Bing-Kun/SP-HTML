什么是 Spark？

Spark 是初（网名）编写的一款模拟操作系统展示页面，它并非真正的操作系统，无法在电脑上实际运行程序，仅作为一个创意展示项目供大家观赏。Spark 主要由模拟的安装界面和图形操作界面组成。第一代 Spark 并不叫作 Spark 1.0.0.0，而是被称作 dictionary 1.0，开发者也许是希望它能查阅到很多信息吧……不过可惜的是 dictionary 1.0 的源代码已经找不到了……

为什么要编写 Spark？

Spark 的灵感来源于开发者在编程社区中闲逛时，看到了一个名为 Windows 10 的模拟桌面编程作品，于是便有了制作一款模拟系统展示的想法。Spark 最早编写于 2021 年，每一代展示页面都随着上一代的迭代而更新，视觉效果也变得更加流畅和精致。随着越来越多的 Java 插件和云存储功能的引入，在 Spark 4 发布结束后，首次出现了联网的一批代表性展示作品，主要以多个用户移动不同的区块来进行图形交互操作。在 Spark 5 之后，模拟系统就支持了联网功能，Spark 也就顺理成章地进入了互联网时代。Java 插件是一个非常有用的东西，它允许开发者通过 Java 的引入来进行更多有趣的展示操作，比如 Spark 5 的一项功能——"聊天"就采用了"汉字转换英文 1.0"。
